Widget Integration Guide

To add the Danfoss Product Assistant to your website, simply add this script tag before the closing </body> tag:

<!-- Danfoss RAG Chat Widget --> <script src="https://your-domain.com/static/danfoss-chat-widget.js" data-api-url="https://your-api-domain.com" data-primary-color="#E2000F" data-title="Danfoss Product Assistant"> </script>

Configuration Options

  • data-api-url - The URL of your Danfoss RAG API backend
  • data-primary-color - Brand color for the widget (default: #E2000F)
  • data-title - Title shown in the chat header
Note: Make sure your API backend is configured with proper CORS settings to allow requests from your distributor website domain.
